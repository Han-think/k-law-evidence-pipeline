# Search Ranking Policy

Legal-source search should prefer reliability and relevance over volume.

## Ranking factors

1. Source reliability grade
2. Exact title or article match
3. Jurisdiction match
4. Date relevance
5. Source type match
6. Query term coverage
7. User mode fit

## Source priority

A-grade official sources should rank above secondary materials when relevance is comparable.

## Date handling

Legal sources can change over time. Search results should expose:

- current version
- effective date
- retrieved date
- historical version if available

## Result grouping

Group results by type:

- statutes and articles
- cases and decisions
- administrative rules
- agency materials
- secondary context

## Result warnings

Show warnings when:

- only secondary sources are found
- date is missing
- source authority is unclear
- article text is not retrieved
- jurisdiction is unsupported

## Rule

A search result is not a legal conclusion. It is a source candidate until verified.
