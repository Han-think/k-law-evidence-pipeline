# Complaint Drafting Assistant

The drafting assistant helps users prepare structured drafts. It does not replace professional review or official filing decisions.

## Drafting stages

```text
case intake
→ timeline
→ material list
→ issue candidates
→ source candidates
→ claim/request candidates
→ draft outline
→ review checklist
```

## Civil complaint outline

A civil complaint draft should be built from structured fields:

- document title
- parties
- requested outcome
- factual background
- basis for responsibility
- damage or harm summary
- connection between facts and requested outcome
- materials list
- source candidates
- attachment list
- review checklist

## Public complaint or petition outline

A public complaint draft should include:

- recipient agency
- requester information placeholder
- issue summary
- timeline
- requested action
- related official sources
- attached materials
- contact and privacy note

## Demand letter outline

A demand letter draft should include:

- recipient
- sender placeholder
- factual summary
- requested action
- requested deadline
- material list
- source candidates
- reservation of rights wording

## Internal memo outline

For public officials or businesses:

- issue summary
- facts known
- facts unknown
- source candidates
- compliance or administrative risk
- recommended next check
- recordkeeping note

## Draft safety labels

Every draft should include one status:

- draft only
- needs source verification
- needs material support
- ready for professional review

## Rule

The assistant should transform structured information into draft text. It should not decide final legal strategy, claim amount, or filing action.
