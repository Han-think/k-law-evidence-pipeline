# Source Policy

This project must separate official sources, trusted public materials, secondary materials, and user statements.

## Source classes

### A: Official direct source

Examples:

- statute text from an official law database
- court decision from an official source
- administrative decision from an official agency
- official notice or regulation text

Use for legal citation candidates.

### B: Official explanation or public agency material

Examples:

- official guide pages
- agency FAQ pages
- public institutional explanation materials

Use for context, not as a replacement for the original rule text.

### C: Trusted public institution material

Examples:

- public legal aid materials
- university or research institute material
- official civic education material

Use as supporting explanation.

### D: Secondary source

Examples:

- news article
- blog post
- commentary
- community post

Use for background or event tracking only.

### X: Unverified

Examples:

- user memory
- unsourced claim
- copied text without citation

Use only as a lead for verification.

## Rule

Drafts may mention source candidates, but formal citation should use A-grade sources whenever possible.

If an official source cannot be retrieved, the system must mark the result as unverified instead of generating missing source text.
