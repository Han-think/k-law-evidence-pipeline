# MVP Scope

The first version should be small, practical, and safe.

## Included in MVP-1

- local case note format
- timeline template
- evidence list template
- Korean document templates
- simple privacy masking
- official source adapter skeleton
- MCP server skeleton
- citation metadata schema
- missing material checklist

## First case categories

- personal data leak and compensation preparation
- telecom service interruption or unfair handling dispute
- e-commerce refund or contract dispute
- basic small civil claim preparation
- public petition or complaint draft

## Not included in MVP-1

- automatic court submission
- final legal advice
- guaranteed legal outcome prediction
- automatic claim amount decision
- paid case management
- storage of real case files in the repository

## Development order

1. Documentation and templates
2. Case schema
3. Privacy masking
4. Timeline and evidence list builder
5. Korean official source adapter
6. MCP server tools
7. Draft generator
8. Review checklist
